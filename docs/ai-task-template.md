# AI task template

## Inputs
- Exact Figma node URL:
- Node role:
  - library family
  - exact variant
  - screen instance
- Existing repo components to reuse:
- What should be built:
- What should NOT be built:

## Workflow
1. Inspect the exact full Figma node URL
2. Identify reuse vs new build
3. Propose public API
4. List visible states/variants
5. Propose Storybook coverage
6. Stop for approval before implementation unless explicitly told to continue

## Storybook expectations
- Playground
- Variant/state stories
- Parity story if applicable
- No invented states or layout behavior beyond the source of truth
